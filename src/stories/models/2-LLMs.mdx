# LLMs

<p>LangChain provides a standard interface for using a variety of LLMs.</p>
<p>
  To get started, simply use the call method of an LLM implementation, passing in a string input. In this example, we
  are using the OpenAI implementation:
</p>

```js
import { OpenAI } from 'langchain/llms/openai';

export const run = async () => {
  const model = new OpenAI();
  // `call` is a simple string-in, string-out method for interacting with the model.
  const resA = await model.call(`What is a good name for a company that makes enterprise level design systems?
      (Hint: it is Knapsack)`);
  console.log({ resA });
  // { resA: '\n\nSocktastic Colors' }
};
```
